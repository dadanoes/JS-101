Setelah mengakses DOM, kita bisa mengubah konten, atribut dan gaya elemen tsb

- Mengubah konten: - element.textContent (mengubah konten text)
                   - element.innerHTML (mengubah konten HTML)
- Mengubah atribut: - element.setAttribute(attribute, value)
                    - element.getAttribute(attribute)
- Mengubah gaya:  - element.style.property = value
