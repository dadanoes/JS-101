"Event Handling" berfungsi untuk menangkap, merespon interaksi pengguna
seperti klik, gerakan mouse, input keyboard dll

Event adalalah sebuah tindakan atau kejadian didalam web
contohnya:
    - klik tombol
    - pengisian form
    - scrolling
semua event tersebut bisa ditangkap dan ditangani JavaScript

Event Listener adalah mekanisme untuk menangkap event dan menjalankan fungsi tertentu ketika event tersebut terjadi
    - onClick (inLine HTML)
    - addEventListener (DOM API/JS)

jenis-jenis event yang umum digunakan:

1. Mouse event:
    - click : elemen diklik
    - dblclick: elemen didouble klik
    - mouseover: mouse berada diatas elemen
    - mouseout: mouse meninggalkan elemen
2. Keyboard event:
    - keydown: tombol keyboard ditekan
    - keyup: tombol keyboard dilepas
    - keypress: tombol keyboard ditekan dan dilepas
3. Form event:
    - submit: form disubmit
    - change: nilai form berubah
    - focus: elemen mendapatkan fokus
    - blur: elemen kehilangan fokus
    - input: nilai form berubah
4. Window event:
    - load: halaman selesai dimuat
    - resize: ukuran window berubah
    - scroll: pengguna melakukan scroll
