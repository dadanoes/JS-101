------------------- OOP ------------------
OOP adalah sudut pandang bahasa pemprograman yang berkonsep "objek"
istilah yang wajib dimengerti yaitu: Object dan Class

Object adalah data yg berisi field/properties/attribute dan method/function/behaviour

Class : - Blueprint atau cetakan untuk membuat Object
        - berisi semua properties dan function yang dimiliki oleh Object
        - setiap Object selalu dibuat dari Class
        - dan sebuah class bisa membuat Object tanpa batas
