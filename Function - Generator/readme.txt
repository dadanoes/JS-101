Function Generator adalah function yang digunakan untuk membuat data Generator
Generator adalah data yang bisa di iterasi seperti array
untuk membuatnya harus menggunakan * (bintang) setelah kata function
untuk mengembalikan data ditiap iterasi, gunakan kata kunci yield diikuti datanya

generator sifatnya Lazy
- jika datanya belum diambil dari generator, maka yield sebelumnya tidak dieksekusi
