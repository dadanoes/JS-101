DOM travelsal adalah proses menjelajahi node di DOM tree untuk menemukan elemen tertentu
dapat berpindah ke elemen orang tua, anak, saudara menggunakan berbagai properti dan metode

Properti DOM Transversal:
- parentNode: Mendapatkan elemen orang tua
- childNode: Mendapatkan node list dari semua anak elemen
- firstChild/lastChild: Mendapatkan elemen anak pertama/terakhir
- nextSibling/previousSibling: Mendapatkan elemen saudara berikutnya/sebelumnya
