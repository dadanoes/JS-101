menambahkan elemen baru ke DOM

- createElement(tagName): membuat elemen baru
- appendChild(childNode): menambahkan elemen sebagai anak dari elemen lain
- insertBefore(newNode, referenceNode): menambahkan elemen sebelum elemen referensi

menghapus elemen dari DOM

- removeChild(childNode): menghapus elemen anak
- remove(): menghapus elemen itu sendiri
