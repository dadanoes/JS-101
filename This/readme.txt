kata kunci 'this' adalah referensi ke object milik siapa
penggunaan terbaik 'this' adalah ketika membuat object method

lokasi-lokasi 'this':
    - object method: object pemilik function
    - global scope: global object (window)
    - function: global object (window)
    - function strict: undefined
    - event: elemen yang menerima event(OOP)
