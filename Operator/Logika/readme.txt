Operator Logic (Logika operator)

~ Digunakan untuk menggabungkan ekspresi logika dan menghasilkan nilai boolean (true atau false)

&& (AND = menghasilkan true jika kedua operand true)
|| (OR = menghasilkan true jika salah satu operand true)
! (NOT = negasi nilai boolean) (negasi adalah penyangkalan atau peniadaan) (kebalikan nilai variabel)
