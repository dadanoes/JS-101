Destructing berarti meng-ekstrak value dari array atau object, lalu memasukan value tsb kedalam variabel

- [] untuk destructing array
- {} untuk destructing object

atau

membongkar value dari array atau object kedalam variabel-variabel

fungsi dari destructing yang paling penting adalah ketika:
- mengambil value dari array atau object tanpa harus melakukan pengambilan satu persatu
- mengembalikan value dari function
- membuat code yang lebih ringkas dan mudah dibaca
