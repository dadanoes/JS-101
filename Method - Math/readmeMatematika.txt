Math Object

math.PI untuk nilai π (3.14159...).
math.E untuk nilai konstanta Euler (2.718...).

mengembalikan nilai mutlak = math.abs()
pangkatkan = math.pow()
akar kuadrat = math.sqrt()
akar kubik = math.cbrt()
nilai maksimal = math.max()
nilai minimal = math.min()

// ⁡⁣⁣⁢Pembulatan Angka⁡ //
math.round() untuk membulatkan ke atas atau ke bawah
math.ceil() untuk membulatkan ke atas
math.floor() untuk membulatkan ke bawah
math.trunc() untuk membulatkan ke nol
