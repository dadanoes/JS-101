                        Template Literal
Template literals adalah fitur yang diperkenalkan di ES6 (ECMAScript 2015)
Menyediakan cara lebih modern dan fleksibel untuk bekerja dengan string.

Dengan Template Literal kita bisa:
melakukan string interpolation (menyisipkan variabel/ekspresi ke dalam string)
membuat string multi-baris
menyertakan ekspresi langsung di dalam string.

Interpolasi String (String Interpolation) = menyisipkan ekspresi (seperti variabel) langsung ke dalam string menggunakan sintaks ${}.
String yang mencakup beberapa baris tanpa perlu menggunakan karakter khusus seperti \\n.
