========== Function ===========

- blok program yang berjalan saat kita panggil
- function dapat digunakan berulang kali

basic function:
- Function Declaration (Deklarasi Fungsi)
- Function Parameter/Function Argumen = - berfungsi untuk mengirim informasi yg ingin kita panggil
                                        - parameter ditulis didalam ()
- Function Return Value:
    - secara default function tidak mempunyai Value
    - kata kunci 'return' dapat memasukan value dalam function
