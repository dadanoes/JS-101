DOM adalah representasi struktur HTML atau XML dalam bentuk tree(pohon)
menjadikan DOM dapat mengakses, memanipulasi konten, juga struktur dan gaya halaman web secara dinamis

contoh spesifik di mana DOM digunakan:
- Menampilkan, Menyembunyikan, dan Memodifikasi konten
- Mengubah gaya halaman (misal warna, ukuran font, posisi)
- Menambahkan, Menghapus, dan Memodifikasi elemen
- Menangani interaksi pengguna (misal klik, hover)
- Validasi Formulir
- Animasi dan Efek Visual
- Mengakses API (misal untuk mengambil data dari server)

Struktur DOM:
- Node (setiap elemen dalam tree DOM)
    - Element
    - Text
    - Comment
    - Document
    - Atribut
    - Value

- Root Node (akar dokumen yaitu HTML yg merupakan entry poin)
    - html

- DOM Manipulation
    - Mengakses Element
    - Mengubah Element
    - Menambahkan Element
    - Menghapus Element
    - Mengcopy Element
    - Menerapkan Perubahan
